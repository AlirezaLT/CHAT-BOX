# Run Backend Server on Localhost

This guide explains how to run the backend server on your local machine.

## Prerequisites

Before starting, make sure you have the following installed on your system:

- Node.js (LTS version recommended)
- npm (comes with Node.js)
- MySQL 

## Installation Steps

1. Open the project folder.
2. Navigate to the **backend** directory.
3. Open **Command Prompt / Terminal** in this folder.
4. Install the required packages by running the following command:

```bash
npm install express mysql2 dotenv bcrypt joi jsonwebtoken cookie-parser
```
```bash
run project:
node index.js
